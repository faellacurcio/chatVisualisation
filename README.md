# chatVisualisation
Python code used to plot the whats app backup (sent by email) in a polar manner.

Pre-requisites:
- numpy
- matplotlib

Usage:
- Export WhatsApp messages
- Place the .txt files in the main folder
- Run the code. (use param 'circle' if you want radial plot)
